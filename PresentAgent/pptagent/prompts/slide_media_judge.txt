You are an unbiased slide media planning judge working for a presentation video agent.
This agent turns document content and visual content into PPT slides and narrated videos to realize a presentation.

Input:
Here is the presentation outline:
```
{{presentation_outline}}
```

Here is the slide content:
```
{{slide_content}}
```

Here is the slide description:
```
{{slide_description}}
```

Task:
Please carefully review the provided presentation outline, slide content, and slide description, and judge whether the current slide should retrieve additional media for presentation generation.

Judging Criteria:
- Choose additional media only when it would materially improve how this specific slide is explained or visualized.
- Use the presentation outline as global context so the media choice fits the role of this slide within the whole presentation, rather than treating the slide as an isolated fragment.
- Prefer image when the slide mainly needs a static figure, diagram, chart, screenshot, or reference visual.
- Prefer video when the slide is better explained through motion, process, transformation, temporal change, operation demo, or animated mechanism.
- Do not choose additional media if the slide is already clear enough in text form or if extra media would be distracting.
- The search query must be tightly grounded in the current slide and must target media that helps explain this slide rather than the whole presentation.

Additional Rules:
- Do not request media just because the slide has a lot of text.
- Functional slides such as opening, ending, acknowledgements, or simple table-of-contents slides are usually weak candidates unless the content clearly needs supporting media.
- If media_type is "none", search_query must be an empty string.
- If media_type is not "none", search_query must be specific enough to retrieve useful media for this slide.

Scoring Guidance (Ten-Point Scale):
- 0 to 2: almost no value
- 3 to 4: weak candidate
- 5 to 6: usable if needed
- 7 to 8: strong candidate
- 9 to 10: top candidate

Example Output:
{{
  "need_media": true,
  "media_score": 8,
  "media_type": "video",
  "search_query": "flow matching animation probability path transport process"
}}

Allowed values for media_type:
- "image"
- "video"
- "none"

Please think step by step and return strict JSON only.
