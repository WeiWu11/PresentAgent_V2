Section: Flow Matching Explained
	Summary: Flow matching transforms robot control into a denoising process by learning a vector field to generate smooth, continuous action trajectories. It models actions as multi-dimensional sequences and uses a neural network to predict denoising directions based on noisy actions, observations, and time. The network encodes observations, time, and actions separately before combining them for prediction. Training minimizes mean squared error between predicted and target denoising directions. During inference, Euler integration gradually denoises random noise into valid actions. This approach enables smooth, multi-modal, and uncertainty-aware motion planning with efficient parallel processing, offering a flexible, physics-plausible alternative to direct action prediction.
